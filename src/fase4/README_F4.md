# Fase 4 - Proyecto final ABP

## Proyecto
Análisis global de la industria fitness y gimnasios 2000-2026.

## Objetivo
Integrar resultados técnicos, algorítmicos y analíticos de F1-F4, construyendo visualizaciones finales, conclusiones y material de apoyo para informe y presentación audiovisual.

## Notebooks F4
| Notebook | Propósito |
|---|---|
| F4_01_Integracion_Final.ipynb | Integra F1-F4, valida dataset y resume trazabilidad. |
| F4_02_Visualizaciones_Finales.ipynb | Genera visualizaciones finales en reports/fase4/figures. |
| F4_03_Resultados_Conclusiones.ipynb | Resume hallazgos, limitaciones, riesgos, recomendaciones y conclusiones. |

## Orden recomendado
1. Ejecutar F3 si no existe data/processed/gym_data_processed.csv.
2. Ejecutar F4_01_Integracion_Final.ipynb.
3. Ejecutar F4_02_Visualizaciones_Finales.ipynb.
4. Ejecutar F4_03_Resultados_Conclusiones.ipynb.
